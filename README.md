# Sistema de Vendas Desktop (Tkinter)

Sistema de vendas desenvolvido em Python com interface gráfica (Tkinter), voltado para pequenos negócios e revendedores que precisam de uma solução simples para controle de produtos, estoque e vendas.

## 📌 Descrição

O sistema utiliza um controle de acesso por login, redirecionando o usuário para dois perfis diferentes:

- Cliente  
- Administrador  

Cada perfil possui funcionalidades específicas e interface própria.

---

## 👤 Perfil Cliente

- Visualização de produtos disponíveis  
- Exibição de descrição, preço e estoque  
- Adição de produtos ao carrinho  
- Cálculo automático do valor total da compra  
- Opção de finalizar compra  

---

## ⚙️ Perfil Administrador

- Edição de produtos (nome, descrição, preço e estoque)  
- Controle de estoque em tempo real  
- Visualização de informações dos produtos  
- Relatório de vendas contendo:
  - Data  
  - Produto  
  - Quantidade  
  - Cliente  

---

## 💡 Objetivo do Projeto

Este sistema esta sendo desenvolvido com foco em revendedores e pequenos comerciantes, permitindo uma melhor organização das vendas e controle do negócio de forma prática e acessível.
